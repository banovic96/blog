cd


Komanda cd (Change Directory) koristi se za promenu aktivnog direktorijuma. Po potrebi, možemo koristiti i apsolutne putanje (sledeća naredba ima isti efekat kao i prethodna): Najprostiji slučaj podrazumeva prebacivanje u poddirektorijum (trenutno aktivnog direktorijuma), pa ukoliko trenutno aktivni direktorijum (~) sadrži direktorijum folder, dovoljno je da uputimo sledeću komandu:

$ cd /home /korisnik /folder

cd - home direktorijum

cd .. vracanje na prethodni direktorijum

cd / root direktorijum postaje tekuci

cd- vracanje na direktorijum koji je bio tekuci pre zadnje komande
