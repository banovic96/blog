cd


cd komanda menja trenutni radni direktorijum, na direktorijum koji navodimo kao agument, kucanjem:


$ cd doc1

Tekuci direktorijum je doc1


$ cd ..

Vracanje na prethodni roditeljski direktorijum


$ cd

Vracanje na Home direktorijum


$ cd -

Vracanje na direktorijum koji je bio tekuci pre zadnje cd komande


$ cd ../doc1/doc2

Relativno kretanje po stablu
