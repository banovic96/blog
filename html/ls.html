ls



Komanda ls prikazuje sadrzaj aktivnog direktorijuma. Ako pozovemo komandu ls bez dodatnih opcija,
dobicemo prost listing aktivnog direktorijuma.


Lista se sadrzaj aktivnog direktorijuma

input:

$ ls -a

output:

. .. doc doc1 doc2



Lista se sadrzaj direktorijuma u dugackom formatu

input:

$ ls -l

output:

-rw-rw-r-- 1 user 6 мар 8 10:15 doc
-rw-rw-r-- 1 user 6 мар 8 10:15 doc1
drwxrwxr-x 2 user 4096 мар 8 10:12 doc2



U listi direktorijumi su oznaceni sa / a fajlovi sa *

input:

$ ls -F

output

doc doc1 doc2/
