cat man


Osnovna namena komande cat jeste spajanje datoteka. Takodje se cesto koristi za ispis sadrzaja datoteka
u terminalu. Komanda cat nije pogodna za prikaz velikih fajlova.


Da bismo videli sadrzaj datoteke koristimo komandu cat

input

$

cat doc1
output

Hello world



Ako zelimo da vidimo broj reda

input

$

cat -n doc1
output

1 Hello world


Prikaz sadrzaja svih tekstualnih datoteka u direktorijumu

input

$

cat *.txt
output

hello
world



Primer spajanja dve datoteke

input

$

cat dat1 dat2
output

good day
people


Kreiranje datoteke 'dat1'

$

cat dat1
